FAQ

O launcher é gratuito? +

Sim — TkLauncher é distribuído gratuitamente. Confira a licença no repositório do GitHub.

Possui Login Microsoft? +

Sim — o launcher suporta autenticação via Microsoft (Xbox Live) para contas oficiais.

Posso jogar offline? +

Sim, há suporte a perfis offline para jogar sem conexão.

Instala automaticamente? +

Sim — downloads e instalação de arquivos necessários são automatizados para facilitar a experiência.

Como atualizar? +

Use o recurso de atualizações automáticas no launcher ou baixe a última release na página de Releases.
